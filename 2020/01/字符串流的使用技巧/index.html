前提概要

这篇的灵感来源于刷题的崩溃，可爱开心的刷着pat结果发现不会字符串流根本刷不下去(不是真的刷不下去，你可以用字符数组代替，但是确实很麻烦)， 所以被迫学习。不得不说，C++给你封装好了很多东西，会C的都知道，C对字符串很不友好，但是C++就不一样了，字符串很是强大，再配合stl模板， 刷题真的很快，所以在此之前我总结一篇关于字符串流的教程帮助大家。

istringstream

istringstream是由一个string对象构造而来，istringstream类从一个string对象读取字符，看下基本使用方法 构造字符串流的时候，空格会成为字符串参数的内部分界， 例子中对a,b对象的输入”赋值”操作证明了这一点，字符串的空格成为了整型数据与浮点型数据的分解点，利用分界获取的方法我们事实上完成了字符串到整型对象与浮点型对象的拆分转换过程。 上面的例子需要注意.str()为istringstream的成员函数，类似的还有很多，我们下期再介绍

3.字符串流+stack+reverse函数

题目要求：(请注意，输出最后面是没有空格的哦，且输出的单词中间是要有空格的) 输入: “hello world” 输出: “dlrow olleh” 这个题目就是上面的题目进化版，每个单词的字母是倒序，并且单词整体也是倒叙，所以我们可以很快的想到先对字母进行倒序，然后再将单词倒序 字母倒叙上面已经说过了，单词倒叙就用stl模板的stack来实现，看代码
* 补充一点，关于reverse函数的原型，介绍一下，有助于你们理解这个函数的实现原理

全部代码

#include <iostream>
#include<stack>
#include<sstream>
using namespace std;

int main()
{
	string tmp("hello world");
	string p;
	bool start = true;
	//getline(cin, tmp);
	stringstream ss;
	stack<string> s;	
	cout << tmp<<endl;
	ss << tmp;
	while (ss >> tmp)
	{
			reverse(tmp.begin(), tmp.end()); //string 可以调用reverse函数,string的表现像一个vecor<char>
			p = p + tmp+' ' ;
			cout << tmp;
	}
	ss.clear();
	ss << p;
	while (ss >> p)
	{
		s.push(p);
	}
	while (!s.empty())
	{
		if (start)
		{
			cout << s.top();
		}
		else
		{
			cout << " " << s.top();
		}
		start = false;
		s.pop();
	}
	system("pause");
	return 0;
}
