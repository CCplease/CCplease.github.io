基本思想

快速排序大家肯定都听过，可爱在学这个排序的时候老师说是冒泡排序的进化(晕~，当时学完对这句话一脸懵逼，感觉跟
冒泡没啥关系)。快排的基本思想就是：二分法。把数列分成两半，再递归直到只剩2个数或者1个数。所以总结快排的大致
方法就是：二分+递归==快排。那么逻辑上是如何的呢？1、确定主元(也叫基准数 or pivot)。2、找出主元后如何分出两
个独立的子集。请看图(紫色的为主元)

核心代码(指针交换法)1

  • 那么何谓指针交换，我们来看看
    我们首先选定基准元素Pivot，并且设置两个指针left和right，指向数列的最左和最右两个元素： 接下来是第一次循环，从right指针开始，把指针所指向的元素和基准元素做比较。如果大于等于Pivot，则指针向左移动； 如果小于Pivot，则right指针停止移动，切换到left指针。
    在当前数列中，1<4，所以right直接停止移动，换到left指针，进行下一步行动。
    轮到left指针行动，把指针所指向的元素和基准元素做比较。如果小于等于Pivot，则指针向右移动；如果大于Pivot，则 left指针停止移动。
    由于left一开始指向的是基准元素，判断肯定相等，所以left右移一位。
    由于7 > 4，left指针在元素7的位置停下。这时候，我们让left和right指向的元素进行交换。 接下来，我们进入第二次循环，重新切换到right向左移动。right先移动到8，8>4，继续左移。由于2<4，停止在2的位置。 切换到left，6>4，停止在6的位置。 元素6和2交换。 进入第三次循环，right移动到元素3停止，left移动到元素5停止。 元素5和3交换。 进入第四次循环，right移动到元素3停止，这时候请注意，left和right指针已经重合在了一起。 当left和right指针重合之时，我们让Pivot元素和left与right重合点的元素进行交换。此时数列左边的元素都小于4，数 列右边的元素都大于4，这一轮交换终告结束。 大致流程讲完了，看下代码吧

#include<cstdio>
using namespace std;

int getPartition(int nums[], int low, int height);

void quicksort(int arr[], int left, int right)
{
	// 递归结束条件：left大等于right的时候
	if (left >= right) {
		return;
	}
	// 得到基准元素位置,而且数列已经被分成2部分了
	int pivotIndex = getPartition(arr, left, right);
	// 根据基准元素，分成两部分递归排序
	quicksort(arr, left, pivotIndex - 1);
	quicksort(arr, pivotIndex + 1, right);
}

int getPartition(int nums[], int low, int height)
{
	int keyVal = nums[low];
	while (low < height)
	{
		while (low < height&&nums[height] >= keyVal)
			height--;
		nums[low] = nums[height];
		while (low < height&&nums[low] <= keyVal)
			low++;
		nums[height] = nums[low];
	}
	nums[low] = keyVal;
	return low;
}


void Quick(int arr[], int n)
{
	quicksort(arr, 0, n - 1);
	
}
int main()
{
	int arr[] = { 3,4,8,38,4,-12,93,74,23,-48 };
	int i;
	cout << "原始数组：" << endl;
	for (i = 0; i < 10; i++)
		cout << arr[i] << "\t";

	Quick(arr, 10);
	cout << endl << "快排后的数组：" << endl;
	for (i = 0; i < 10; i++)
		cout << arr[i] << "\t";

	getchar();
	return 0;

总结

以上就是快排所有的代码，让我们来讨论一下快排的注意点。

  • 关于pivot的选择
    首先，我上面的所有快排主元都是选择数列最左边的元素，这个办法好不好呢？说实话还可以，但是有一种情况就不行了，即数列为有序 数列，因为每次递归你都得重新遍历。所以很多教材或者网上都会对pivot放在单独得函数先进行处理一下，我例举其中一种(方法有很多) 但是对于初学者你把最左边的那个数当作主元也还可以(而且挺快的，实测)，毕竟一般有序数列不太出现，影响不太大。而且加了这个主 元的确定会增加遍历的时间，但是作为学习敲代码很有好处

  • 最好 & 最坏的情况
    我们都知道任何排序都有最好最坏的情况，就快排而言，它的时间复杂度为：T(N)=O(Nlog·N)。当主元每次都在中间，换言之不用遍历元素， 每轮循环都在正确的位置为最好情况。但如果让我们对一个数组从大到小排序，但数组顺序正好是从小到大的呢？思考一下，根据快排的原理， 我们每一轮的排序仅仅只是确定了主元的位置，而却没有分成两个独立的子集，这就很闹心了，这不就变成了冒泡排序了么(我当时想到这里 才觉得快排是冒泡的进化，晕~，也不知道是不是)。那么这样时间复杂度：T(N)=O(N^2),不懂？将下面的图跟基本原理的图进行比对。

  • 历史小知识(看着玩的)
    快排是由C.A.R.Hoare(东尼·霍尔，Chrarles Antony Richard Hoare)在1960年提出，并且后续又有许多人对此进行了优化，如果感兴趣 可以看看《算法导论》第七章或者其他相关书籍，我在这里给大家一个思路，本章所说的快排以及小伙伴平时学的快排大都是只有一个主元， 其实还可以有两个，三个主元(3个以上的主元就没必要了，速度随着主元的增多开始下降了)，其中两个主元的快排(Dual-pivot)是最快的。 比三个主元，一个主元都要快，且快排(dual-pivot)的排序速度仅次于基位排序，这是一个相当快的排序算法哦！有兴趣的小伙伴可以尝试。

#include<iostream>
#include<cstdio>
#include<cmath>
#include<ctime>
#include<algorithm>
using namespace std;
#define N 100000

int partition(int arr[], int low, int high);


void quicksort(int arr[], int left, int right)
{
	// 递归结束条件：left大等于right的时候
	if (left >= right) {
		return;
	}
	// 得到基准元素位置,而且数列已经被分成2部分了
	int pivotIndex = partition(arr, left, right);
	// 根据基准元素，分成两部分递归排序
	quicksort(arr, left, pivotIndex - 1);
	quicksort(arr, pivotIndex + 1, right);
}


int partition(int arr[], int i, int j) {
	// 取第一个位置的元素作为基准元素
	int pivot = arr[i];
	int left = i;
	int right = j;
	while (left != right) {
		//控制right指针比较并左移
		while (left<right && arr[right] > pivot) {
			right--;
		}
		//控制right指针比较并右移
		while (left < right && arr[left] <= pivot) {
			left++;
		}
		//交换left和right指向的元素
		if (left < right) 
		{
			int p = arr[left];
			arr[left] = arr[right];
			arr[right] = p;
		
		}
	}
	//主元和指针重合点交换，主元不是pivot
	int p = arr[left];
	arr[left] = arr[i];
	arr[i] = p;
	

	return left;
}

void Quick(int arr[], int n)
{
	quicksort(arr, 0, n - 1);
}

void bubble(int arr[], int n)
{
	int i = NULL;
	for (i = 0; i < N; i++)
	{
		for (int j = 0; j < (N - i); j++)
		{
			if (arr[j] < arr[j + 1])
			{
				int t = NULL;
				t = arr[j];
				arr[j] = arr[j + 1];
				arr[j + 1] = t;
			}
		}
	}
}

int main()
{
	int *p = new int[N];	//	原数组，会变
	int *cpy = new int[N];	//用来暂时存放的数组，因为设计多个算法

	int i = 0;
	clock_t begin;
	srand((unsigned)time(NULL));
	for (i = 0; i < N; i++)
		p[i] = rand() % 1000000;

	memcpy(cpy, p, sizeof p);
	begin = clock();

	Quick(p, N);
	cout << "指针交换快排的速度：" << clock() - begin << "毫秒" << endl;

	bubble(cpy, N);
	cout << "冒泡排序的速度：" << clock() - begin << "毫秒" << endl;

	
	getchar();
	return 0;
}
