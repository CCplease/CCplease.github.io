HUGO 个人博客搭建

前提提要

  • 首先你要学会计算机网络，算法，服务器搭建等相关知识（he-tui 根本不用，会认字就行）
  • github这个东西是一定要有的，我们的代码全部放在这里，相当于蹭一下github的服务器
  • LeanCloud账户（评论系统，挺好玩的，可拓展性好）
  • markdown是一定要学会的，已经是一个很大众化的工具了，markdown-教程
  • 如果你想打理好这个博客，html的基本语法也可以了解一下哈，因为markdown本身兼容markdown的语法
  • 所有的操作最好使用vscode进行(别问why，问就是方便)，强大插件、终端和控制台，让你爱不释手

  • 最重要的：每个下载的东西最好都放在除C盘以外的盘符且给文件夹取好名字，否则上午下载，下午就忘

    安装git

    自行安装git版本工具，可爱大发慈悲给个链接吧git安装教程

    安装hugo

    　　windows10下安装hugo，可以参照Hugo官方手册的方法，这里讲一个相对简单稳定的方法。 以下安装hugo转自呆神的博客

  • 下载hugo程序压缩包：前往https://github.com/gohugoio/hugo/releases下载和自己系统版本相符合的hugo程序压缩包。

  • 解压到某个文件夹中（路径不要有中文，而且自己要记得文件夹的路径），最好是不常改动的文件夹下边，以防文件被误删或者丢失。

  • 添加hugo到系统环境变量PATH中

    1. 找到“系统环境变量”的设置位置，在开始菜单的搜索栏搜索环境变量
    2. 添加系统环境变量，依此：点击环境变量，找到系统变量中的path，点击编辑，然后点击新建，在使用浏览按钮选中文件夹，即可使用hugo。（选中到hugo.exe所在的文件夹即可，不需要选中hugo.exe）

  1. 接下来，为了万无一失，还是要检查一下hugo是否安装完成。以管理员方式打开cmd命令窗口，然后输入以下指令：

    hugo version

　　如果得到如下响应，（即显示版本信息），说明安装成功，接下来就可以玩转hugo了。

创建并配置hugo站点

以下的操作均在我们刚才调出的bash终端里面执行

  • 创建站点命令，选择一个文件夹(一个新的目录，跟之前下载hugo的文件夹最好分开，比如：d:\hugo_file)，在其中存放站点的相关文件， 文件路径不要有中文 

    hugo new site d:/hugo_file（这个不固定）

下载并配置站点主题（以tranquilpeak主题为例）

在任意的文件目录下使用以下命令重新配置windows IP配置，并刷新DNS缓存：

ipconfig/flushdns

　　在根目录下使用CD命令来到themes文件夹下，并且在tranquilpeak主题下很容易找到clone命令如下：

cd themes
git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git tranquilpeak

#以下为注释说明，可以在clone的整句命令后加上你要clone到的文件夹名字，默认是hugo-tranquilpeak-theme
#可是默认的文件夹名称实在是太长了，为了方便使用，可以在命令后边加上空格然后指定为一个短点儿的文件夹名字，比如这里的tranquilpeak

　　一般情况下可以很快完成clone，如果速度还是很慢，可以尝试重新修改hosts，或者采用拉取最近版本，然后再拉去全部版本内容，命令如下：

git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git tranquilpeak --depth 1
#等待这条命令走完后访问该主题的文件夹
cd tranquilpeak
#然后clone全部内容
git fetch --unshallow
